# SPRING OS — Direction Engine™

SPRING OS es un sistema para convertir intención en dirección.

No genera ideas sueltas. No improvisa contenido.
Toma una prioridad, una frecuencia real y un tono, y devuelve un plan ejecutable.

El motor funciona con una llamada directa a IA generativa (Gemini), sin frameworks intermedios ni agentes externos.
La salida está estructurada, validada y lista para usar.

Lo que produce:

– Dirección estratégica en 60 segundos
– Próximos pasos ordenados
– Pilares de contenido
– Calendario mensual
– Piezas iniciales listas para publicar
– Kit visual básico (paleta y reglas)

Está diseñado para evitar fantasía operativa.
Cumplir > fantasear.

Run
pip install -r requirements.txt
streamlit run app.py
